# ALTEN SHOP API

## build the project
```sh
mvn clean install

```


## launch the project:

```sh
mvn spring-boot:run
  or  
mvn spring-boot:run -Dspring-boot.run.profiles=local

windows Powershell:
mvn spring-boot:run "-Dspring-boot.run.profiles=local"
```


## Run the project via docker compose

```sh
docker compose up
